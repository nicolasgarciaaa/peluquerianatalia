# Peluquería Natalia Unisex

Este es el sitio web de **Peluquería Natalia Unisex**, una peluquería en Burgos que ofrece cortes de pelo, barbería y peinados. En nuestra página encontrarás información sobre nuestros servicios, precios, horarios y ubicación.

## Contenido

- **Sobre Nosotros**: Información acerca de los servicios y la experiencia que ofrecemos.
- **Precios**: Tarifas de nuestros servicios.
- **Horario**: Los horarios de apertura y cierre.
- **Ubicación**: Dirección de nuestra peluquería y mapa interactivo.

## Tecnologías usadas

- **HTML**: Estructura básica de la página.
- **CSS**: Estilos para una experiencia visual atractiva.
- **JavaScript**: Funciones interactivas, como la navegación suave a las secciones.
